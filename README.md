# NFT Collection Generator

Professional-grade tool for creating generative art NFT collections from layered image components. Automatically generates unique combinations of layers based on customizable rarity weights and produces official ERC721-compliant metadata.

## Features

- **Layer-Based Generation**: Combine multiple image layers (backgrounds, bodies, heads, accessories, hats, etc.) to create unique NFTs
- **Rarity Control**: Fine-tune the frequency of each trait variation using customizable rarity weights
- **Batch Generation**: Generate entire NFT collections (100s or 1000s of images) in minutes
- **ERC721 Metadata**: Automatic creation of official NFT metadata files for blockchain deployment
- **Flexible Output**: Configure image dimensions, quality, and output locations
- **Trait Management**: Define realms (style categories), groups (trait types), and individual variations
- **Visual Editor**: Intuitive UI for managing projects and previewing layer combinations

## How It Works

1. **Organize Layers**: Structure image layers into a 3-level hierarchy:
   - **Realm**: Style category (e.g., "Retro Art")
   - **Group**: Trait type (e.g., "Background", "Body", "Head")
   - **Variations**: Individual images for each trait (e.g., 15 different backgrounds)

2. **Set Rarity**: Assign rarity weights to control how often traits appear:
   - Higher rarity value = more frequent in collection
   - Mix common and rare traits for variety

3. **Configure Project**: Set output size (512Г—512, 1000Г—1000, etc.), total NFT count, and metadata details

4. **Generate**: Click Generate to create:
   - Unique composite PNG images (one per NFT)
   - Individual JSON metadata files
   - Combined collection database

## Getting Started

### Creating a New Project

1. Launch the app: `NFTGenerator.exe`
2. Click **"New Project"**
3. Name your collection
4. Click **"Add Folder"** and select your layers directory
5. The app will load all realms, groups, and layer variations
6. Adjust rarity values for each layer
7. Configure output settings
8. Click **Generate**

### Project File Format

Projects are saved as `.nftc` files containing:
```json
{
  "ProjectName": "My NFT Collection",
  "TotalItems": 100,
  "Settings": {
    "OutputDirectory": "path/to/output",
    "OutputSize": "512,512",
    "ResizeAlgorithm": 1
  },
  "Overlays": [ /* layer hierarchy */ ]
}
```

## Output

For each generated NFT collection, the app creates:

- **Images**: `token_0.png`, `token_1.png`, etc. (composited from layers)
- **Metadata**: `token_0.json`, `token_1.json` (ERC721-compliant)
- **Database**: `collection.json` (all tokens and traits)
- **Hashes**: SHA256 checksums for each token

### Example Metadata File
```json
{
  "name": "Retro NFT #0",
  "description": "Retro pixel art NFT generated by NFT Collection Generator",
  "image": "ipfs://QmYourImageHash/token_0.png",
  "attributes": [
    { "trait_type": "Background", "value": "Dark Blue" },
    { "trait_type": "Body", "value": "Pink" },

  ]

```

## Requirements

- **Windows 10+** (or compatible OS with .NET Framework)
- **ImageMagick.NET**: For image processing and composition
- **Newtonsoft.Json**: For JSON serialization

## Settings & Customization

Within the app you can configure:
- **Output Directory**: Where generated files are saved
- **Image Size**: Dimensions for final NFT images
- **Resize Algorithm**: Quality of image scaling (Nearest Neighbor, Bicubic, etc.)
- **Token ID Range**: Starting token ID for your collection
- **Metadata**: Base descriptions, IPFS addresses, reveal status

## Technology Stack

- **Language**: C# (.NET Framework)
- **UI**: Windows Forms
- **Image Processing**: ImageMagick.NET
- **Serialization**: Newtonsoft.Json
- **Build**: MSBuild

## Generating Demo Layers

A Python script is included to generate demo layer images automatically:

### Script: `generate_demo_layers.py`

**Features:**
- Creates 75 retro pixel art PNG images (512Г—512)
- 5 layer groups Г— 15 variations each
- Includes backgrounds, bodies, heads, accessories, and hats
- Automatically organized in proper folder structure

**Usage:**

```powershell
# Navigate to project directory
cd c:\path\to\nft_collection_generator

# Run the script
python generate_demo_layers.py
```

**Output:**
The script creates:
```
DemoNFTAssets/
```

**Requirements:**
- Python 3.7+
- PIL/Pillow: `pip install pillow`

### Using Generated Layers

1. Run `generate_demo_layers.py` to create demo images
2. Open the app and load `Retro_Demo_Project.nftc`
3. Click **Generate** to create 100 unique NFTs from the demo layers

You can also modify the script:
- Change `BASE_SIZE` to adjust image dimensions
- Edit color palettes (`BACKGROUND_COLORS`, `BODY_COLORS`, etc.)
- Adjust layer types in `layers` dictionary
- Customize pixel sizes and patterns
